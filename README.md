# Experiment 6 - Handle Forms and Validations in Frontend

This project is a React-based web application that demonstrates
form handling and validation using Material UI (MUI) components.

It is created as part of Experiment-6 for frontend development practice.

---

## 📌 Objective

To design and develop a responsive form using Material UI
with built-in and custom validations.

---

## 🚀 Features

- Text Fields (Name, Email, Password)
- Email Format Validation
- Password Length Validation
- Radio Buttons (Gender Selection)
- Checkbox (Terms & Conditions)
- Disabled Submit Button until Agreement
- Error Messages
- Gradient UI Design
- Responsive Layout
- Advanced Styling

---

## 🛠️ Technologies Used

- React JS
- Material UI (MUI)
- JavaScript
- HTML5
- CSS3
